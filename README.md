功能分解圖：
![](https://i.imgur.com/qYbizew.jpg)

二手車保養履歷系統的需求分析簡述如下：
1. 車商建立二手車資料
2. 储存現有二手車資料記錄或刪除錯誤資料
3. 消費者可透過此系統查詢二手車所有的保養資料
4. 車商運用此系統對保養資料進行統一管理
5. 系統記錄所有保養過程
